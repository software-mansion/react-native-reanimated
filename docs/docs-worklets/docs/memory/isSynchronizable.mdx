# isSynchronizable

Asserts whether the provided value is a [Synchronizable](/docs/memory/Synchronizable).

You can pass a generic type argument to specify the expected value type held by the Synchronizable.

## Reference

```typescript
import { isSynchronizable } from 'react-native-worklets';
const synchronizable = ...; // type: unknown

console.log(isSynchronizable(synchronizable)); // logs true if `synchronizable` is a Synchronizable.

if (isSynchronizable<number>(synchronizable)) {
  // Here, TypeScript knows that `synchronizable` is of type `Synchronizable<number>`
  synchronizable.setBlocking(42);
}
```

<details>
<summary>Type definitions</summary>

```typescript
function isSynchronizable<TValue = unknown>(
  value: unknown
): value is Synchronizable<TValue>;
```

</details>

## Arguments

### value

The value to be checked.

## Remarks

- `isSynchronizable` can be called from any JavaScript Runtime.
