<Badges version="0.10.0">
  # isBundleModeEnabled
</Badges>

Checks whether [Bundle Mode](/docs/bundleMode) is active.

As of 1.0.0, Bundle Mode is the only mode of running worklets, so this function always returns `true` on native platforms and `false` on the Web. It's kept mainly for compatibility.

## Reference

```typescript
import { isBundleModeEnabled } from 'react-native-worklets';

console.log(isBundleModeEnabled()); // true on native platforms
```

  <summary>Type definitions</summary>

### Returns

`true` on native platforms, `false` on the Web.
